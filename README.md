# Agile Retrospective Toolkit

> A ready-to-use collection of retrospective formats, facilitation guides, and an interactive HTML tool — built from 5+ years of running retros across eCommerce, Banking, Oracle EBS, and Airlines teams.

---

## What's inside

| File | What it is |
|------|------------|
| [`retro-formats-guide.md`](./retro-formats-guide.md) | When to use which retro format and why |
| [`facilitator-playbook.md`](./facilitator-playbook.md) | Full facilitation guide — anti-patterns, timing, team dynamics |
| [`retro-health-tracker.md`](./retro-health-tracker.md) | How to measure if your retros are actually working |
| [`retro-toolkit.html`](./retro-toolkit.html) | Interactive browser tool — run a retro without any external platform |

---

## The interactive tool

Open `retro-toolkit.html` in any browser — no install, no login, no Miro account needed.

It lets you:
- Pick a retro format
- Add sticky notes per category
- Vote on items
- Export action items as a summary

---

## Philosophy

A retrospective that produces no action items is a meeting, not a retro. Every format here is designed to end with named owners and committed actions — not just a list of feelings.
